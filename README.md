# Earth-Moon Lagrange System Simulation

A high-fidelity 3D interactive simulation of the Earth-Moon Lagrange points, featuring the Bellevistat industrial station at L4.

## Features
- Real-time Earth-Moon orbital mechanics
- All 5 Lagrange points (L1-L5)
- Bellevistat space station at L4
- Interactive camera controls
- Mobile-optimized performance
- Toggle controls for triangle, orbits, cargo routes, gravity field

## Controls
- **Drag** - Rotate camera
- **Scroll** - Zoom
- **Click** - Select celestial body/station
- **1-4** - Camera presets
- **T** - Toggle triangle
- **L** - Toggle Lagrange points
- **O** - Toggle orbits
- **C** - Toggle cargo routes
- **G** - Toggle gravity field

## Running Locally
```bash
npm install
npm start
```
